h1

Back To Top

Heading Level 1. Applies settings to all h1 blocks. The browser will search for h1 fist when prioritizing what content to show.

<h1> Favorite Movies </h1>

h2

Back To Top

Heading Level 2. Applies settings to all h2 blocks. Default is an order of magnitude smaller than h1 elements.

<h2> Favorite Action Movies </h2>

h3

Back To Top

Heading Level 3. Applies settings to all h3 blocks. Default is an order of magnitude smaller than h2 elements.

<h3> Favorite Spy Movies </h3>

h4

Back To Top

Heading Level 4. Applies settings to all h4 blocks. Default is an order of magnitude smaller than h3 elements.

<h4> Actors </h4>

h5

Back To Top

Heading Level 5. Applies settings to all h5 blocks. Default is an order of magnitude smaller than h4 elements.

<h5> Favorite Period Movies </h5>

h6

Back To Top

Heading Level 6. Applies settings to all h6 blocks. Default is an order of magnitude smaller than h5 elements.

<h6> Favorite War Movies </h6>

p

Back To Top

Paragraph. Applies settings to all body copy distinguished by the preceding number. (ie. p1, p2, p3 etc.)

<p> Lorem ipsum dolor sit amet consectetur adipisicing elit. Voluptatibus aliquam iusto numquam explicabo ad cum, dolorum provident perferendis aut voluptas vitae nihil dolores, sequi eum facere tenetur illum itaque corrupti. </p>

ul

Back To Top

Unordered List. To organise list items into a bulleted list.

<ul> <li> Pirates of the Caribean </li> </ul>

ol

Back To Top

Ordered List. Organizes list items into a numbered list.

  • reversed- numbers are descending. (9,8,7,6...)
  • start- sequence will begin on a certain number
  • type- specifies type of marker in the list.

<ol> <li> Pirates of the Caribean </li> ;</ol>

li

Back To Top

List Item. Distinguishes List Items.

<li> Pirates of the Caribean </li>

  • value- Specifies number in list.

a

Back To Top

Anchor. Links to another location on page or on a different page.

  • href= ""- designates the location that the anchor links to.

<a href= "https://www.imdb.com/title/tt0325980/"> Pirate of the Caribean </a>

img

Back To Top

Image. links and inserts an image from stylesheet or another location on the web.

  • src- source of image (url, or folder that is hosting the file).
  • alt- What will be displayed if the image cannot load, or screenreader technology is being used

<element> This is an example of how the element is written in html </element>

html

Back To Top

Contains the head and body. This tag communicates to the text editor that your document type is HTML, and the tags and commands that follow are to funtion in HTML.

<html> <head> <title> Kade's Favorite Movies </title> </head> <body> <h1> Favorite Movies </h1> </body> </html>

head

Back To Top

Contains locations of stylesheet and links any elements on other pages or sites. Thought of as the data about the data to be used in the coming document. Also generally contains the title of the document

<head> <title> Kade's Favorite Movies </title> </head>

link

Back To Top

Generally used in the head to link external stylesheets or elements from other sources.

  • rel- Specifies the relationship between the current document and the linked document.
  • href= - Specifies the location of the linked document.
  • type- Specifies the media type of the linked document

<element> This is an example of how the element is written in html </element>
